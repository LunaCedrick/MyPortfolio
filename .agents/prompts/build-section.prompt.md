Before coding:
1. Analyze the current structure
2. Reuse existing styles
3. Preserve spacing consistency
4. Avoid duplicated CSS
5. Keep code beginner-readable
6. Use semantic HTML
7. Maintain responsiveness

- Match existing visual language
- Use modular CSS
- Keep JavaScript minimal
- Comment all codes
