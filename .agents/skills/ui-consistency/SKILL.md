---
name: ui-consistency
description: Maintains visual consistency across the portfolio
---

Ensure:
- Consistent spacing
- Consistent typography
- Consistent hover states
- Consistent card styling
- Consistent red accent usage
- Consistent animation timing
- Font Awesome icons visually replace text placeholders in About, Contact, and Footer
- Favicon uses the same crimson/dark portfolio identity
- Project carousel preserves existing card visuals, hover effects, spacing, and modal behavior
- Project modal created-date styling matches the existing muted label and crimson accent language
